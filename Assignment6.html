  1. For one continuous variable:
  • Density
df %>% ggplot()+
  geom_density(mapping = aes(x = lead_time))

  • Histogram
df %>% ggplot()+
  geom_histogram(mapping = aes(x = lead_time))

  • Boxplot
df %>% ggplot()+
  geom_boxplot(mapping = aes(x = lead_time))

  • Barplot
